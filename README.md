# Marketplace Matching Dashboard
# [Открыть дашборд](https://serioio321.github.io/marketplace-monitoring-dashboard/#overview)

Интерактивный дашборд для мониторинга качества сопоставления товаров в маркетплейсе. Проект сделан как прототип для ВКР и показывает динамику ключевых метрик на синтетических данных.

## Назначение
Дашборд помогает отслеживать покрытие каталога, точность сопоставления, риск деградации и качество данных по департаментам и типам моделей сопоставления.

## Данные
В проекте используются синтетические данные, структурно основанные на реальных данных маркетплейса: сохранены состав полей, логика связей и характер метрик, однако сами значения сгенерированы искусственно, чтобы не нарушать NDA и не раскрывать конфиденциальную информацию.

## Стек
- HTML5
- CSS3
- JavaScript
- Chart.js

## Возможности
- выбор периода данных;
- выбор департамента;
- графики динамики покрытия, качества и риска;
- таблица по департаментам;
- светлая и тёмная тема.

## Структура
- `index.html` — интерфейс дашборда;
- `data.json` — синтетические данные;
- `README.md` — описание проекта.

## Запуск
Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

Проект можно также развернуть через GitHub Pages.
