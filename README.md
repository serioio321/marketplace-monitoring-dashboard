# Marketplace Matching Dashboard

Интерактивный дашборд для мониторинга качества сопоставления товаров в маркетплейсе. Проект сделан как прототип для ВКР и показывает динамику ключевых метрик product matching на синтетических данных.

## Назначение
Дашборд помогает отслеживать покрытие каталога, точность сопоставления, риск деградации и качество данных по департаментам и типам matching.

## Данные
В проекте используются синтетические данные, структурно основанные на реальных данных маркетплейса: сохранены состав полей, логика связей и характер метрик, однако сами значения сгенерированы искусственно, чтобы не нарушать NDA и не раскрывать конфиденциальную информацию.

История данных охватывает 3 месяца и обновляется 3 раза в неделю по расписанию MON, THU, SAT в 17:45.

## Стек
- HTML5
- CSS3
- JavaScript
- Chart.js

## Возможности
- выбор периода данных;
- выбор департамента;
- KPI-карточки;
- графики динамики покрытия, качества и риска;
- таблица по департаментам;
- светлая и тёмная тема.

## Структура
- `index.html` — интерфейс дашборда;
- `data.json` — синтетические данные;
- `README.md` — описание проекта.

## Запуск
Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

Проект можно также развернуть через GitHub Pages.
